The Solution
One file. One command.
Agents that travel.
The .agent file — a portable, self-contained bundle that packages everything about an agent. Share it like a file. Run it anywhere.
# You built an agent. Share it.
agentbox pack .
-> my-agent-1.0.0.agent (0.8 KB)

# Your friend receives it. One command.
agentbox install my-agent-1.0.0.agent
agentbox run my-agent -m "Help me with taxes"
-> Agent running. No setup needed.
