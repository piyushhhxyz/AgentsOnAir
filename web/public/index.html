How It Works

Four commands. That's it.

From idea to shared agent in under a minute.

1

Create

Scaffold an agent with a template, system prompt, and knowledge files

agentbox init my-agent
2

Pack

Bundle everything into a single portable .agent file

agentbox pack .
3

Share

AirDrop, email, Slack — send the .agent file however you want

tax-agent-1.0.0.agent
4

Run

Install and run on any machine with one command

agentbox run tax-agent

The Format

One file. Everything inside.

A .agent file is a portable zip containing the manifest, knowledge, and tools. Framework-agnostic. Self-describing.

agent.yaml — the manifest
name: "tax-agent" version: "1.0.0" description: "AI tax advisor — finds deductions, explains tax law" author: "piyush" agent: system_prompt: | You are an expert tax advisor AI. You help with: 1. Understanding tax obligations and deadlines 2. Finding applicable deductions and credits 3. Explaining tax law in plain English model: provider: "openai" name: "gpt-4o-mini" temperature: 0.3 knowledge: - "tax-brackets-2024.md" - "common-deductions.md" metadata: category: "finance" tags: ["tax", "finance", "deductions"]
